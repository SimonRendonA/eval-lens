# EvalLens Logo Assets

## Structure

```
monogram/                  Standalone mark (hexagon + EL)
  monogram-dark.svg          Primary — dark bg, teal hex (vector)
  monogram-light.svg         Light mode — dark fill, muted teal (vector)
  monogram-transparent.svg   No background (vector)
  monogram-inverted.svg      Gradient bg, dark elements (vector)
  monogram-dark-{size}.png   PNG at 512, 256, 192, 128, 64
  monogram-light-{size}.png  PNG at 512, 256, 128
  monogram-inverted-{size}.png PNG at 512, 256, 128

lockup/                    Monogram + "EvalLens" wordmark
  lockup-dark.svg            For dark backgrounds (vector)
  lockup-light.svg           For light backgrounds (vector)

favicon/                   Browser and app icons
  favicon.svg                32px SVG with EL text
  favicon-16.svg             16px, outer hex only
  favicon.ico                ICO with 16+32 embedded
  favicon-32.png / -16.png   PNG fallbacks
  apple-touch-icon.png       180px for iOS

og-image.png               1200x630 Open Graph social preview
```


## Brand colors

- Primary: #00d4aa
- Accent: #00b4d8
- Primary (light): #009d7e
- Gradient: linear-gradient(135deg, #00d4aa, #00b4d8)
